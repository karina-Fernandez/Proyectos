Mi Proyecto VSC

Ruth Karina Fernández Bendezú

Demostración de lo que aprendí en Visual Studio Code

1Configuré Visual Studio Code

Qué aprendí: Las áreas principales de VSC

Lo que hice:
  • Exploré el explorador de archivos (lado izquierdo)
  • Usé el editor principal (centro)
  • Abrí la terminal integrada (abajo)
  • Instalé extensiones (panel de extensiones)
  • Cambié el tema a oscuro

2Usé la Paleta de Comandos

Qué aprendí: Comandos rápidos con Ctrl+Shift+P

Comandos que usé:
Ctrl+Shift+P > Format Document (para formatear este archivo) > Toggle Terminal (para abrir/cerrar terminal) > Git: Commit (para guardar cambios)

3Aprendí Git en la Terminal

Qué aprendí: Comandos básicos de Git

Comandos que ejecuté para este proyecto:
git init git add mi-proyecto-vsc.html git commit -m "Mi primer proyecto VSC" git push origin main

4Usé Emmet para escribir rápido

Qué aprendí: Atajos para generar HTML rápido

Ejemplos que usé en este archivo:
! → Generó toda la estructura HTML básica div.ejercicio*4 → Creó 4 divs con clase "ejercicio" h2>span.numero → Creó h2 con span adentro

5Dominé los Atajos de Teclado

Qué aprendí: Atajos para ser más rápido

Atajos que usé al escribir este código:
  • Ctrl+D: Seleccioné múltiples palabras iguales
  • Alt+↑/↓: Moví líneas de código
  • Ctrl+/: Comenté y descomente código
  • Ctrl+Shift+L: Seleccioné todas las ocurrencias

✅ Resumen: ¡Ya sé usar VSC!

Este archivo HTML demuestra que aprendí a:

  • Configurar y navegar en Visual Studio Code
  • Usar comandos rápidos con la paleta
  • Controlar versiones con Git
  • Escribir código rápido con Emmet
  • Ser más productivo con atajos de teclado
