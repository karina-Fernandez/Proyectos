Ejemplo con Tailwind CSS

Este ejemplo demuestra cómo usar espaciado, tipografía y bordes con Tailwind CSS. La imagen tiene bordes redondeados y un estilo discontinuo, y los elementos están separados con márgenes y padding.

Caja 1

Padding: p-4
Margen entre cajas: space-x-4

Caja 2

Tipografía: text-base
Borde: border, rounded-md

Caja 3

Espaciado vertical: space-y-4
Responsive: flex-col / flex-row
